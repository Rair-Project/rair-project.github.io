About Rair

Rair is experimental reverse engineering framework written in Rust. It is designed to be reliable, fast and flexible. Rair comes with Rust API as well as interactive radare2-like command line interface. Rair is tested on 64-bit Linux, 64-bit Windows and 64-bit macOS.
